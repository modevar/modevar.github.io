The MODEVAR workshop is planned as a half-day online event at **April 29th, 2026**, 13:00-17:00 (CEST). Find a tentative overview on the program below.

* Session 0: Welcome & Overview on State of UVL
    * Short introduction into MODEVAR and UVL
    * Presentation on recent changes to UVL's core libraries
* Session 1: Tool Demos / Presentation of Use Cases
    * Tool demos by participants targeting UVL and variability modeling in general
    * Presentation of participants showcasing different requirements for variability specifications
* Session 2: How to develop UVL as a community effort?
    * Discussion on latest protocol to continuously incorporate community 
    * Presentation of new repository for language extension proposals
* Session 3: UVL Language Roadmap & Open Discussion
    * Discussion of current UVL roadmap
    * Discussion on change suggestions for the language
